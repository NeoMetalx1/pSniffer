# pSniffer (⚠️ Soft is under development, no user friendly)

pSniffer is a C++ tool for traffic analysis, that uses libpcap

## Installation

Arch Linux: 

```bash
sudo pacman -S libpcap make
```
Ubuntu / Debian:

```bash
sudo apt-get libpcap-dev
```
Fedora:

```bash
sudo dnf install libpcap
```

Compiling: 

```bash
git clone https://github.com/NeoMetalx1/pSniffer.git
cd pSniffer && make
```

## Usage

```
sudo ./sniffer
```
